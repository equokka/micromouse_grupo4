# Micromouse

## Building

This command builds to `./bin/main`
```sh
make
```

## Running

Run `./bin/main`

You can build+run:
```sh
make run
```

Then on the client, press <kbd>W</kbd><kbd>A</kbd><kbd>S</kbd><kbd>D</kbd> to move. <kbd>Q</kbd> to quit.

If you need to force-close, use <kbd>Ctrl+C</kbd>.
